INITIAL SETUP:
1. Create database "aluguer_motas" in a MySQL manager software (like phpMyAdmin)
2. Populate it by running "models.py". The database will be populated with default entries for all tables. There will be 2 example clients (Carlos and Ana) - Carlos will be the admin, so he will be able to access the control panel in order to add/edit/delete motorcycles, rentals and add admin privileges to other users)
3. Run an http server (Apache used as the development server) and then run the "app.py" to start the application through the given link
